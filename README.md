# GitHub Actions CI/CD Pipeline Lab

## 📌 Overview
This project demonstrates how to create a basic **CI/CD pipeline using GitHub Actions**.  
The workflow automatically runs when code is pushed or a pull request is created.

---

## 🚀 What is GitHub Actions?
GitHub Actions is a built-in CI/CD tool that helps automate:
- Testing code
- Building applications
- Deploying applications
